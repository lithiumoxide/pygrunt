A minimal Terragrunt wrapper for Python

## Usage
```
from pygrunt import pygrunt as pg

pg.plan('/path/to/terragrunt.hcl')
```

## Installation
```
pip install pygrunt
```

## Requirements
* Terragrunt
* Terraform

## Compatibility
Built with Python 3.8
